A lightweight, responsive CSS framework built with SCSS.

Theming

Create a theme.scss file in your project's src/scss/ directory. Define your colors before importing the framework — all variables use !default so yours take priority.

// theme.scss — Your custom theme
$primary:   #e74c3c;
$secondary: #2c3e50;
$success:   #27ae60;
$danger:    #c0392b;
$warning:   #f39c12;
$info:      #3498db;

$font-family-base: 'Inter', sans-serif;
$border-radius: 0.5rem;

// Import the framework after your overrides
@import 'tina4';

That's it! Every button, alert, badge, form focus ring, and utility class will automatically use your colors.
